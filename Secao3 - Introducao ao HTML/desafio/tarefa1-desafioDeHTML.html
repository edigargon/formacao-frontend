Tarefa: Desafio de HTML

O aluno deve criar uma estrutura do HTML, seguindo as orientações da aula de apresentação

  1. Crie um novo arquivo HTML, nomeie como quiser;
  2. Utiliza a inicialização rápida de estrutura HTML;
  3. Crie um titulo que descreva uma imagem;
  4. Insira a imagem;
  5. Crie um paragrafo que fale mais sobre o que está na imagem;
  6. E por fim uma lista, com três itens com elementos presentes na imagem;
